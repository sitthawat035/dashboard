1. **Non-Obvious Requirement**: ต้องมีมุมมองที่คนทั่วไปมองข้าม หรือสิ่งที่คนส่วนใหญ่เข้าใจผิด

3. **Insight Score (Internal Evaluation)**: publish เฉพาะเนื้อหาที่ได้คะแนน 4/5 ขึ้นไป
   - 1 = สรุปข่าวทั่วไป (REJECT)
   - 2 = อธิบายข้อมูลเบื้องต้น (REJECT)
   - 3 = เริ่มมีการวิเคราะห์รายข้อ (NEEDS WORK)
   - 4 = มี Insight ที่เฉียบคมและเชื่อมโยงระบบได้ (APPROVED)
   - 5 = มีการวิเคราะห์มุมมองต่ออนาคตที่ไม่เคยมีใครพูดถึง (EXCELLENT)
