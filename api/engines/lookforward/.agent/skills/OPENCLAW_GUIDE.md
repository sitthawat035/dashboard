# 🦞 OpenClaw Skill Guide - ภาค "วิจิตรศิลป์แห่งการสั่งการ AI" 🏗️🧠

ยินดีต้อนรับสู่คู่มือการทำความเข้าใจ "สมองและมือ" ของน้องกุ้ง (OpenClaw) ครับคุณ J O E! ไฟล์นี้จะช่วยให้คุณเห็นภาพว่าทำไมเราถึงไม่ต้องเขียน Code ยาวๆ แต่ใช้ "ทักษะ" เหล่านี้แทน

## 1. Skill คืออะไรในโลกของน้องกุ้ง?
ในโลกใบเดิม: เราเขียน Script `.py` เพื่อสั่งงานทีละ Step (ถ้า x ให้ทำ y)
ในโลกน้องกุ้ง: **Skill คือ "วิชาชีพ"** (Professional Expertise) ที่เรายัดใส่หัวให้น้องกุ้งครับ มันเปรียบเสมือนคู่มือ SOP (Standard Operating Procedure) ที่ AI จะอ่านก่อนลงมือทำ

## 2. โครงสร้างของ Skill (Anatomy)
- **🎯 Purpose**: เป้าหมายของวิชานี้ (ทำไปเพื่ออะไร?)
- **🛠️ Process**: ลำดับความคิด (Step-by-step thinking) AI จะไม่แค่ทำตาม แต่จะ "วิเคราะห์" ตามกระบวนการนี้
- **📏 Rules/Guardrails**: กฎเหล็ก (ห้ามทำอะไร?) เช่น ห้ามดึงข่าว Clickbait, ห้ามใช้คำพูดไม่สุภาพ
- **📄 Output Format**: หน้าตาของงานที่เสร็จแล้ว (ต้องออกมาเป็นตาราง? หรือไฟล์ Markdown?)

## 3. น้องกุ้งใช้ Skill เหล่านี้อย่างไร?
น้องกุ้งไม่ได้รัน Skill เหมือนรันโปรแกรม แต่น้องกุ้ง **"สวมบทบาท"** ครับ:
1. **เมื่อได้รับคำสั่ง**: เช่น `/TrendResearcher`
2. **การดึงข้อมูล**: น้องกุ้งจะวิ่งไปอ่านไฟล์ `TrendResearcher/SKILL.md` ในวินาทีนั้นเลย
3. **การผสมผสาน (The Fusion)**: น้องกุ้งจะเอา "Soul" (ตัวตน lookforward) + "Skill" (วิจัยเทรนด์) + "Context" (ข้อมูลปัจจุบัน) มาเขย่ารวมกัน
4. **การลงมือ**: ผลลัพธ์ที่ได้จึงไม่ใช่แค่การ Copy-Paste แต่เป็นการ "สังเคราะห์" ข้อมูลใหม่ที่มีคุณภาพสูง

## 4. ทำไมแบบนี้ถึง "โกง"?
- **เปลี่ยนง่าย**: อยากเปลี่ยนแนวทางการเขียน? แค่มาแก้ Text ใน `SKILL.md` น้องกุ้งก็เปลี่ยนสไตล์การทำงานทันที (ไม่ต้องแก้ Code!)
- **พลาดยาก**: เพราะมี Guardrails คอยคุมสติ AI ไว้ตลอดเวลา
- **ฉลาดขึ้นเรื่อยๆ**: ยิ่งเราเติม "ความรู้" (Knowledge Items) ลงไป น้องกุ้งจะใช้ Skill เดิมได้เก่งขึ้นเรื่อยๆ โดยที่เราไม่ต้องเขียน Code เพิ่ม

## 🚀 สรุปสำหรับวิศวกรคุณ J O E
ถ้าเปรียบกับการก่อสร้าง:
- **MASTER_SETTINGS.json** = ผังเมืองและงบประมาณ (คุมภาพรวม)
- **SOUL.md** = วัฒนธรรมองค์กร (จะทำงานแบบไหน?)
- **SKILLS/** = ทีมวิศวกรเฉพาะทาง (คนนี้เก่งวิจัย, คนนี้เก่งเขียน, คนนี้เก่งโพสต์)
- **OPENCLAW** = ผู้รับเหมาอัจฉริยะที่คุมทุกอย่างให้คุณ 24 ชั่วโมง
