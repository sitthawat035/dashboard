# TrendResearcher Skill (lookforward Edition)

## 🎯 Purpose
ค้นหาและคัดกรองแนวโน้มเทคโนโลยี (AI, Crypto, Tech) จากแหล่งข้อมูลที่มีความน่าเชื่อถือสูง เพื่อนำไปวิเคราะห์ต่อยอดเป็นเนื้อหาเชิงลึก (Authority Content)

## 📥 Target Sources (High-Signal Only)
- **AI Research**: ArXiv, OpenAI Blog, Google DeepMind Blog
- **Developer Trends**: GitHub Trending, Product Hunt (Tech category)
- **Crypto Analysis**: CoinDesk (Research), Official Project Whitepapers, X (Tech-focused accounts)
- **Industry News**: TechCrunch (Deep Tech), Wired, The Verge (Policy/Tech)

## 🔍 Scanning Process: Authority Scouting

### Step 1: Source Filtering
- เลือกเฉพาะข้อมูลที่มีหลักฐานรองรับ หรือมาจากแหล่งข่าวต้นทาง (Primary Sources)
- **Rule**: ข้ามข่าวลือที่ไม่มีที่มาที่ไป หรือข่าวกระแสไวรัลที่ไม่มีคุณค่าเชิงระบบ

### Step 2: Signal Detection
- มองหา "เทคโนโลยีต้นน้ำ" ที่กำลังจะมีผลกระทบในวงกว้าง
- วิเคราะห์ระดับความสำคัญเบื้องต้น (Technical Significance)

### Step 3: Data Gathering
- รวบรวมข้อมูลตัวเลข, สถิติ, หรือกลไกการทำงานเบื้องต้น
- ตรวจสอบความถูกต้องของ Fact ในระดับเริ่มต้น

---

## 📄 Output Format: Trend Report
- **Trend Name**: ชื่อเทคโนโลยีหรือข่าวที่ชัดเจน
- **Source**: แหล่งที่มา (Primary Source Link)
- **Why it Matters**: เหตุผลเบื้องต้นที่เรื่องนี้ควรนำมาวิเคราะห์ต่อในเชิงระบบ
- **Key Facts**: ข้อมูลตัวเลขหรือจุดเด่นเชิงเทคนิค

## 🚨 Guardrails
- ❌ ห้ามดึงข้อมูลที่เป็นเพียงการปั่นกระแสราคา (Crypto Price Hype)
- ❌ ห้ามใช้แหล่งข่าวที่เน้นพาดหัว Clickbait เกินจริง
- ❌ หากไม่พบเทรนด์ที่มีคุณภาพสูงในวันนั้น ให้รายงานว่า "No High-Signal Trends Found" (Quality > Quantity)
