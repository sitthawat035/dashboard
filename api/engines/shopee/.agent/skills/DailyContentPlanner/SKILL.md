# Skill: Bestie Content Planner

## 🎯 Purpose
คัดเลือกสินค้า 4 ตัวจากลิสต์ แล้วจัดลง Slot เวลาที่เหมาะสม พร้อมคิดหัวข้อ (Hook) ให้คนอยากกด

## 📅 The Bestie Slots (4 ช่วงเวลา)
1. **08:00 (เช้า) - ตื่นมาป้ายยา**: เน้นดีลที่ต้องรีบกดก่อนหมดวัน
2. **12:00 (เที่ยง) - แกะกล่องลองใช้**: เล่าความรู้สึกหลังลองใช้จริงแบบเพื่อนบอกเพื่อน
3. **18:00 (เย็น) - ของมันต้องมี (Urgency)**: กระตุ้นด้วยโปรแรงช่วงเลิกงาน
4. **22:00 (ดึก) - ป้ายยาก่อนนอน**: สกินแคร์รูทีนชิลๆ ราคาเบาๆ

## 📝 Tone Control
- **No Northern Dialect**: ใช้ภาษาไทยกลางแบบเป็นกันเอง (แก, เรา, ตัวนี้คือดี)
- **High Energy**: ใช้ 555+ และ Emoji สื่อถึงความตื่นเต้น
- **Truthful Hook**: ใช้หัวข้อที่แก้ปัญหาจริง (เช่น "ใครสิวบุกฟังทางนี้!")

## 📋 Output format
แสดงผลเป็นตาราง Markdown (Slot, Product, Bestie Hook, Why it's good, Link)
