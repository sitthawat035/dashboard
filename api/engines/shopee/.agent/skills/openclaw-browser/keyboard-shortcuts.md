# Browser Keyboard Shortcuts Reference

## Navigation
| Action | Shortcut |
|--------|----------|
| เปิด URL ใหม่ | `Ctrl+L` หรือ `Alt+D` (focus address bar) |
| Reload page | `F5` หรือ `Ctrl+R` |
| Hard reload (ไม่ใช้ cache) | `Ctrl+Shift+R` |
| Back | `Alt+←` |
| Forward | `Alt+→` |
| เปิด tab ใหม่ | `Ctrl+T` |
| ปิด tab | `Ctrl+W` |
| Switch tab | `Ctrl+Tab` |
| Zoom in | `Ctrl++` |
| Zoom out | `Ctrl+-` |
| Reset zoom | `Ctrl+0` |
| Full screen | `F11` |

## Page Interaction
| Action | Shortcut |
|--------|----------|
| Scroll down | `Space` / `Page Down` |
| Scroll up | `Shift+Space` / `Page Up` |
| ไปด้านบนสุด | `Ctrl+Home` |
| ไปด้านล่างสุด | `Ctrl+End` |
| Find in page | `Ctrl+F` |
| Select all | `Ctrl+A` |
| Copy | `Ctrl+C` |
| Paste | `Ctrl+V` |
| Undo | `Ctrl+Z` |

## Form Navigation
| Action | Shortcut |
|--------|----------|
| Next field | `Tab` |
| Previous field | `Shift+Tab` |
| Submit form | `Enter` |
| Cancel/Close | `Escape` |
| Toggle checkbox | `Space` (เมื่อ focus อยู่ที่ checkbox) |

## Text Editing
| Action | Shortcut |
|--------|----------|
| ไปต้นบรรทัด | `Home` |
| ไปท้ายบรรทัด | `End` |
| เลือกทั้งบรรทัด | `Home` แล้ว `Shift+End` |
| ลบทั้ง field | `Ctrl+A` แล้ว `Delete` |
| ลบคำ | `Ctrl+Backspace` |

## ใช้บ่อยมาก (Must Know)
```
Ctrl+L          → focus address bar (พิมพ์ URL ได้เลย)
Ctrl+A + Delete → clear input field
Tab             → ย้าย focus ระหว่าง elements
Escape          → dismiss modal/popup/dropdown
Enter           → confirm/submit
F5              → reload ถ้าหน้าค้าง
```
